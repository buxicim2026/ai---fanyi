💡 使用帮助

📹 视频/音频文件处理流程

  1. 上传视频或音频文件
  2. 配置语音识别 API（推荐 Azure 或 Whisper）
  3. 系统自动提取音频并识别语音
  4. 使用 Qwen AI 翻译成中文
  5. 导出带时间轴的字幕文件

🔑 语音识别 API 获取

  • Azure: Azure Portal → Speech Services
  • Google: Google Cloud → Speech-to-Text
  • Whisper: OpenAI Platform → API Keys
  • 阿里云: 阿里云智能语音

⚠️ 注意事项

  • 大文件处理时间较长，请耐心等待
  • 建议单个文件不超过 100MB
  • 视频文件会自动提取音频轨道
  • API 调用会产生费用，请注意配额
